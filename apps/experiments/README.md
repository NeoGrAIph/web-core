# `apps/experiments`

Песочница для экспериментов и быстрых прототипов.

Правила:
- **не** деплоится в ArgoCD;
- не хранить прод‑критичный код/секреты/контракты;
- можно использовать для проверки гипотез (например UI‑паттернов, интеграций, миграционных стратегий).

Локально:
- `pnpm --filter @synestra/experiments dev`
