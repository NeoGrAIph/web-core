# Миграции (Payload)

Эта директория зарезервирована под миграции Payload для Postgres.

Контекст:
- Шаблон `ecommerce` upstream (BETA) подразумевает активное развитие схемы (orders/products/variants и т.п.).
- В Kubernetes миграции должны выполняться до rollout приложения, чтобы избежать “битого” runtime.

Правила:
- Миграции создаются/применяются через Payload CLI.
- “Seed/demo” эндпоинты из upstream шаблонов по умолчанию не переносим в prod‑контур.
