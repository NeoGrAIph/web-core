# `@synestra/cms-blocks`

Пакет для общих content blocks (Payload `blocks`), которые переиспользуются:
- в корпоративном сайте,
- в лендингах,
- в (части) storefront.

Основание:
- Payload template `website` активно использует “сборку страниц из блоков” (через Pages/Blocks).

Правила:
- блоки должны быть максимально переиспользуемыми;
- не тянуть app‑специфичные зависимости;
- держать совместимость между apps (и помнить про Tailwind 3 vs 4, если UI будет на Tailwind).
