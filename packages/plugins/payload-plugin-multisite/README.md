# `@synestra/payload-plugin-multisite`

Плагин‑заготовка для мультисайтовости/разделения сайтов на уровне Payload.

Зачем:
- со временем может потребоваться переиспользовать одну CMS‑часть между несколькими “витринами”,
  или держать общий набор коллекций, но разделять домены/контент.

Важно:
- сейчас плагин ничего не меняет — это каркас под будущую реализацию;
- при развитии плагина обязательно фиксировать миграционные последствия и контракты env vars.
